FX.php is a PHP class that interacts with FileMaker data using the XML output by Server or Server Advanced (for FileMaker 7+) or the Web Companion (for FileMaker 5/6). From there, you can use PHP's diverse and powerful functions alongside FX.php to manipulate erroryour data, publish it on the internet, run scripts in FileMaker, and so on.
